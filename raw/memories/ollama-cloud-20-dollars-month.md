# Ollama Cloud — $20/month for VPS Users

## Context

From [[summary-bart-slodyczka-gemma4-searxng-openclaw|Gemma 4 + SearXNG + OpenClaw]]. See also [[ollama]].

## Content

Ollama offers a $20/month cloud plan designed for users on VPS or devices too small to run models locally. The plan covers typical OpenClaw daily usage and provides access to large models including Kimi K2.5, GLM 5, and MiniMax M2.7. Also enables running Gemma 4 31B with the full 256K context window via cloud instance.
