# MiniMax M2.7 Cost and Performance Claims vs Opus 4.6

## Context

From [[summary-damian-malliaros-minimax-m27|MiniMax M2.7 + Claude Code]] by [[Damian Malliaros]]. See [[MiniMax M2.7]] and [[Open-Source Model Integration]].

## Content

Claims made by Damian Malliaros (unverified against independent benchmarks in this wiki):

- **Benchmarks**: Outperforms Opus 4.6 on SWE benchmarks (coding and logic tasks)
- **Cost**: ~$1 per hour of use; approximately **20x cheaper** than Opus 4.6
- **Speed**: Approximately **3x faster** response time than Opus 4.6

Pricing model: subscription-based coding plans rather than pay-per-token:
- **Starter**: $10 (Damian built a full ClickUp clone + had usage remaining)
- **Max High Speed**: $80/month — 15,000 model requests per 5 hours (~unlimited coding)
- **Ultra High Speed**: $150/month — 30,000 requests per 5 hours

Compared to Claude Code Max at $200/month with usage limits, the $80 tier offers effectively unlimited throughput at lower cost.

**Note**: These are the creator's claims. Independent benchmark verification not yet in this wiki.
