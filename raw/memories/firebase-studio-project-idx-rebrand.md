# Firebase Studio = Project IDX Rebrand + Gemini + Genkit

## Context

From [[summary-worldofai-firebase-studio|WorldofAI's Firebase Studio video]]. See also [[firebase-studio]].

## Content

**Launched at Google Cloud Next 2025** as the rebrand of Project IDX. Combines into a single cloud agentic dev environment:

- **Firebase** — backend/database/auth
- **Genkit** — Google's GenAI framework (Python, Go, RAG, multimodal orchestration)
- **Gemini** — chat-driven inline edits, file generation, agent commands
- **Vertex AI** — access to Gemini, Imagen 3, and open-source models (Llama, Mistral)
- **GitHub + Cloud Run** integrated deployment

**Form factor**: cloud IDE that combines Bolt-style natural-language prototyping with a Cursor-style agentic editor in one product. 60+ templates spanning web (Angular, Next.js, React), mobile, Python, databases, backend.

**Free tier**: 3 workspaces (30 for dev program members), $300 in credits for new users. Note: WorldofAI hit a "high demand" banner during testing — feature gating may differ from marketing.
