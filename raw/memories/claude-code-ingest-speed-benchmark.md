# Claude Code Ingest Speed Benchmark

## Context

From [[summary-karpathy-10x-claude-code|Karpathy 10x'd Claude Code]] — [[Nate Herk]]'s live demonstration. See [[Claude Code]].

## Content

Nate Herk's measured ingest times using Claude Code for wiki maintenance:
- **36 YouTube transcripts**: ~14 minutes total
- **One long article**: ~10 minutes

These times cover full ingest — reading the source, creating a source-summary page, creating or updating entity/concept pages, updating the index, and cross-referencing.
