# Devin

An autonomous software engineering agent by Cognition. Designed to handle complete software engineering tasks end-to-end — not just code suggestions, but full workflows: research, plan, code, test, debug, deploy.

## What It Is

Devin is positioned at the more autonomous end of the coding agent spectrum. Where [[Cursor]] is an AI-enhanced IDE and [[Claude Code]] is an agent you direct from the terminal, Devin is designed to be given a task and execute it with minimal human input.

## Context

One of the first AI products marketed explicitly as a "software engineer" rather than a coding assistant. Its launch attracted significant attention (and skepticism) for its autonomy claims.
