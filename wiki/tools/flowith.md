# Flowith (Neo)

Multimodal "infinite execution agent" running on a persistent canvas. Each plan node has its own context and analysis surface, supporting parallel sub-tasks and revisiting. Pitches a 10M-token context window, 30+ native integrations, and a Knowledge Garden for per-project RAG. Free tier includes agent mode + Claude 4 Opus backend.

## Key features

- **Infinite canvas** — plan nodes persist; agent can branch and revisit
- **10M token context** (claimed)
- **Knowledge Garden** — upload-and-RAG knowledge bases scoped to a flow
- **Multimodal generation** — text, image, video, 3D models, voice/podcast
- **Deployable artifacts** — one-click publish to Flowith subdomain or custom domain

## Demo highlights

- AI learning website with light/dark mode and quizzes (one prompt)
- New York City "virtual museum" with 3D models + narrated podcast + interactive web deploy
