# Obsidian Web Clipper

A browser extension that converts web articles to markdown and saves them directly into an [[Obsidian]] vault. Essential tool for getting sources into the `raw/` folder of the [[LLM Wiki Pattern]].

## Usage Tip

Set the default save location to `raw/` in the extension's options so clipped articles land in the right folder automatically.
