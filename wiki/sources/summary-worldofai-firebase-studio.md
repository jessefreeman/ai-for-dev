# Google's Firebase Studio: FULLY FREE AI Code Editor!

**Channel:** [[worldofai|WorldofAI]]
**Format:** YouTube video
**Published:** 2025-04-09
**Sponsor:** None disclosed

## Summary

WorldofAI walks through Firebase Studio, Google's rebrand of Project IDX launched at Cloud Next 2025. Unifies Firebase, Genkit, and Gemini into a single cloud agentic dev environment for building full-stack apps via natural-language prompts. 60+ templates, GitHub + Cloud Run integration, free tier with three workspaces. Demo: a finance dashboard generated from a single prompt with backend commands auto-executed.

See [[firebase-studio]] for the entity page.

## Key Points

- **Rebrand of Project IDX** with major upgrades — agentic chat, Gemini integration, expanded Genkit (Python, Go, RAG, multimodal orchestration)
- **Vertex AI access** brings Gemini, Imagen 3, and open-source models ([[llama|Llama]], Mistral) into the same environment
- **60+ templates** spanning web (Angular, Next.js, React), mobile, Python, databases, backend, miscellaneous
- **Cloud IDE + prototyping** combined — Bolt-style natural-language app gen plus a Cursor-style agentic editor in the same product
- **Free tier**: three workspaces (30 for dev program members), $300 credits for new users
- **Agentic features**: inline edits, file change previews, terminal command execution, error recovery
- WorldofAI hit a "high demand" banner during testing — couldn't access the prototype dashboard initially
- **Demo**: built a finance app (income/expenses/stocks visualization) from a single natural-language prompt; agent self-corrected a failed backend startup command
