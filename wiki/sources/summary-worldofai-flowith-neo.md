# Flowith Neo: First-Ever Infinite Agent

**Author / channel:** [[worldofai|WorldofAI]]
**Format:** video
**Source:** [Original](../../raw/archive/Flowith%20First-Ever%20Infinte%20Agent%21%20INSANELY%20Powerful%20AI%20Agent%20Can%20Automate%20and%20Build%20Anything%21.md)
**Published:** 2025-06-18

## Summary

Flowith Neo is a multimodal "infinite agent" — an [[manus]]/[[devin]] competitor that runs on an infinite canvas where each plan node has its own dedicated context and analysis. Claims a 10M-token context window and 30+ tool integrations out of the box. Demoed building a responsive AI learning website (Claude 4 Opus backend) and a New York City "virtual museum" with 3D models, narrated podcast, and interactive web deploy from a single prompt. Free tier available.

## Key Points

- **Infinite canvas execution model** — each plan step is a node on a persistent canvas with its own analysis surface; the agent can spawn parallel sub-tasks, retain per-node context, and revisit nodes. Differs from linear-chain agents like OpenManus or Manus.
- **10M token context window** — claimed; positions Flowith above Gemini 2.5 Pro (1M) and matches the pitch from [[minimax-m1]] / Kimi K2 in scale.
- **30+ native tool integrations** — Knowledge Garden (custom RAG knowledge base per project), web search, image/video gen, model comparison, prompt enhancement.
- **Agent Mode** — toggle that switches Flowith from chatbot to autonomous executor; required for "infinite" task flows.
- **Deployable artifacts** — "Publish" button deploys generated apps to a Flowith subdomain or custom domain. Demo showed a working AI course platform with light/dark mode and quizzes.
- **Free tier** — agent mode and Claude 4 Opus available without payment at the time of recording.
