# 5 (Real) AI Agent Business Ideas For 2025

**Author / channel:** [[simon-hoiberg|Simon Høiberg]]
**Format:** video
**Source:** [Original](../../raw/archive/5%20%28Real%29%20AI%20Agent%20Business%20Ideas%20For%202025.md)
**Published:** 2025-03-24

## Summary

Simon Høiberg lays out five concrete monetization paths for AI builders in 2025: selling [[n8n]] workflows, knowledge-based RAG chatbots, custom LoRA models, vibe-coded prototypes, and AI automation agencies. The framing is realistic and rejects the "vibe everything" hype — vibing is a prototyping phase, not a methodology. Pairs with [[ai-automation-workflows|Nate Herk's automation playbook]] and the [[saas-death-spiral]] thesis.

## Key Points

- **Sell n8n workflows** — two models: (a) export the JSON workflow file as a one-time-purchase template; (b) expose the workflow via authenticated webhook so buyers call it as an API on a per-call/subscription basis (resources stay on your account, so price accordingly).
- **Knowledge-based RAG chatbots** — package an expert's knowledge as a paywalled chatbot (Høiberg pitches his own 8base product). Critical anti-pattern: never put a knowledge base in a custom GPT system prompt — easily extracted via prompt injection.
- **Custom LoRA models** — fine-tune Flux, Stable Diffusion, or DeepSeek for specialized output (e.g., HDRIs for Blender, high-CTR YouTube thumbnails). Deploy on Replicate, sell API access via direct key or proxy.
- **Vibe coding as prototyping** — pixels.io's Peter Levels game and Vibe Sail referenced as proof of monetizable indie hacks. Høiberg explicitly rejects "vibe everything" as a methodology — use it to test waters, then switch to intentional execution.
- **AI automation agency** — done-for-you implementation for businesses that won't buy templates. Liam Ottley cited as the existence proof that this category is now real.
