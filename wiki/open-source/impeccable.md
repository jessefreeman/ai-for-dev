# Impeccable

An open-source Claude Code skill optimized for frontend design. Ships with 17 commands to improve AI-generated UIs beyond the default "purple gradient" vibe-coded aesthetic.

## Commands

Notable commands include:
- **distill** — simplify an overly complex UI in one go
- **colorize** — apply brand colors
- **animate** — add motion
- **delight** — add polish and uniqueness

## Why It Exists

A common complaint about vibe-coded apps: they all look the same. Impeccable addresses this by giving Claude Code a structured set of design-focused operations that can be applied incrementally to make UIs look distinctive and professional.

## Compared to Google Stitch

- **[[Google Stitch]]**: Generate full designs from scratch via voice/text, export design.md
- **Impeccable**: Improve an existing AI-generated UI with targeted commands
