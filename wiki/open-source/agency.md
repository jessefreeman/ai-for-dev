# Agency

An open-source project providing pre-built agent templates for Claude Code, covering standard startup roles: frontend developer, backend developer, security engineer, growth hacker, Twitter engager, and more. Combine multiple agent personas to go from zero to product without manually implementing each personality and skill set.

## How It Works

Each template defines a role-specific agent persona with skills, behavioral constraints, and focus areas. You combine agents together in Claude Code sessions, each bringing their specialty to the project.

## Compared to GStack and Superpowers

All three are Claude Code skill/template systems, but with different philosophies:
- **Agency**: Role-based agent templates (hire a team)
- **[[GStack]]**: YC startup methodology (think like a CEO/CTO/designer)
- **[[Superpowers]]**: TDD engineering workflow (brainstorm→plan→execute→review)
