# Frontend Design Skill

The most-installed Claude Code skill — **277,000 installs** at time of source, the largest install count of any skill the wiki has tracked. Forces Claude to commit to a real design direction before writing any frontend code, instead of defaulting to generic "AI-slop" output.

## What it does

- **Bans certain fonts** (Inter, Roboto, Arial — the classic AI-default trio).
- **Forces Claude to pick a real design direction** before writing a single line of code: bold typography, intentional color palette, layout decisions made up-front.
- **Output**: more opinionated, less "2014 WordPress template" looking UIs. The source shows a before/after on a music-player one-shot build.
