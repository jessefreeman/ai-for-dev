# PDF Processing Skill

Anthropic-official Claude skill for PDF manipulation: read, extract tables, fill forms, merge, split. Source claim: 15 separate apps are reportedly splitting $10M monthly recurring revenue between them on PDF processing alone — there is clear demand for this primitive.

## What it does

- Read PDFs (text + tables)
- Extract tabular data
- Fill PDF forms
- Merge and split PDFs

The source cites an Indie Hacker user who reduced a 90-minute contract review to under a minute using this skill.
