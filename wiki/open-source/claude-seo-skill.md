# Claude SEO Skill

Claude skill for end-to-end site SEO audits. **12 sub-skills.** Crawls a site, validates schema, identifies what's broken, returns a fix list.

## What it does

- Full website audits
- Schema validation
- 12 sub-skills covering different SEO surfaces
- Output is an actionable fix list, not a generic report

The source's pitch: does in seconds what an SEO consultant charges thousands for, and doesn't try to upsell a follow-up call.
