# Brand Guidelines Skill

Encodes a brand's voice, colors, tone, and rules into a Claude skill. Once installed, Claude auto-applies the brand to everything it generates — emails, landing pages, social posts — without needing to be reminded each prompt.

## Why it's interesting

This is the wiki's first plugin entry that operates as **persistent style state for an agent**. Sibling pattern to the [[memory-rules|memory system]] in the wiki's own methodology — both are "stop telling the agent the same thing every conversation."
