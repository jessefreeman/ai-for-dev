# Canvas Design Skill

Claude skill for generating social graphics, posters, and covers from text prompts. PNG / PDF output. The source frames it as "Canva, but you stay inside Claude" — useful for quick social posts and thumbnails, not a full design tool replacement.

> **Naming note:** the source verbally calls this "Canva design," but the title card on screen is "Canvas Design." Recorded under the on-screen name to avoid trademark confusion with the unrelated [[canva]] product.
