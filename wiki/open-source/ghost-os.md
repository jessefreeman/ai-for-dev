# Ghost OS

AI agent that operates **every app on macOS** — not just the terminal or browser, but Keynote, Figma, Photoshop, Slack, and anything else installed. Sees the screen, clicks, drags, types, uses apps the way a human does.

## What it does

- Screen capture as the perception layer
- Native macOS-level click / drag / keystroke as the action layer
- Cross-app workflows (move data between Figma → Keynote → Slack, etc.)
- Source flag: **early stage**. Demo videos described as "genuinely unsettling in a good way."

## Type and platform

- **Type:** CLI / agent runtime
- **Platforms:** macOS only
- **Status:** tracked

## Position vs related "computer use" tools

- **vs [[playwright-mcp]]** — Playwright is browser-only but cross-platform; Ghost OS is macOS-only but covers every native app.
- **vs Anthropic Computer Use / [[managed-agents]]** — Ghost OS runs locally on the user's machine; those are hosted services.
- **vs [[stagehand]]** — Stagehand is browser-agent infrastructure; Ghost OS is full-OS agent infrastructure.
