# Awesome Claude Skills

Curated GitHub list of Claude skills maintained by Travis Vian. **95,000 stars** at time of source — the best community-maintained index of the Claude Code plugin ecosystem.

## What it is

- A README listing notable Claude skills, organized by category
- Regularly updated; community contributions accepted
- The default starting point for "what skills exist?" before hitting the marketplaces

## Why it's tracked here

This isn't a plugin you install — it's a *directory* of plugins you might install. Recorded on the wiki because the [[agent-plugins]] index relies on having one canonical curated source for new entries to flow in from. Awesome Claude Skills is currently that source, alongside the official Anthropic skills repo.
