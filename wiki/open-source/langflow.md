# LangFlow

Visual drag-and-drop builder for AI agent pipelines. **146,000 GitHub stars**, backed by [DataStax](https://www.datastax.com). For people who think better in canvases than code.

## What it does

- Drag components onto a canvas
- Wire them together to build agent pipelines
- No orchestration code required
- Runs the resulting pipeline as a working agent
