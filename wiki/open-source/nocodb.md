# NocoDB

Open-source no-code database platform. Wraps a SQL database (Postgres / MySQL / SQLite) in a spreadsheet-like UI with views, forms, automations, and an API. Free self-hostable Community Edition.

## Position

[[self-hostable-alternatives|Self-hostable alternative]] to **Airtable**. Side benefit per Simon Høiberg: data privacy, since the underlying database lives on your own server.
